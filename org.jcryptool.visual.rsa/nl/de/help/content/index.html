RSA Verschlüsselung

Die Farben der ersten drei Buttons links zeigen den aktuellen Status. Rot bedeutet dieser Schritt wurde noch nicht bearbeitet. Sind die Buttons grün, bedeutet das, diese Task ist erledigt. Wenn die ersten beiden Buttons grün sind, kann die Verschlüsselung schrittweise mit einem Klick auf Start gestartet oder direkt mit einem Klick auf Verschlüsseln abgeschlossen werden. Mit dem Button Zurücksetzen können Sie die Eingaben immer wieder neu durchführen.
